# HAB Website

This repository is a test app for making dashboards using Django

## Backend development workflow

```json
virtualenv env
. env/bin/activate
pip3 install -r requirements.txt
cd backend
python3 manage.py runserver
```

## Frontend development workflow

```json
npm install 			# wait for installation of dependencies
npm start
```

## For deploying

```json
npm run build
```
